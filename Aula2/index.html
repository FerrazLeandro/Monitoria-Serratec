Nesta aula vamos falar um pouco sobre como publicar o html para que outras pessoas acessem por um link, e um pouco sobre o conceito de CI/CD.


CI/CD é um conjunto de práticas para automatizar o desenvolvimento e a entrega de software.

CI (Integração Contínua): Processo de integrar frequentemente as alterações de código em um repositório, com testes automáticos para detectar erros rapidamente.
CD (Entrega Contínua): Processo que garante que o código esteja sempre pronto para ser lançado em produção, com automação do deploy e testes adicionais.
